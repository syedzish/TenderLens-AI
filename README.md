# TenderLens AI

TenderLens AI is a Vercel-ready compliance matrix agent for tender and RFP review.

This public repository is being initialized safely before the application code is added. Secrets, local tooling state, and confidential files are intentionally excluded from version control.

## Security

- Do not commit API keys or environment files.
- Keep `GEMINI_API_KEY` in local/Vercel environment variables only.
- The `confidential/` folder is local-only and must never be committed.

## Deployment Target

- GitHub repository: `TenderLens-AI`
- Hosting: Vercel
- AI model provider: Gemini API
