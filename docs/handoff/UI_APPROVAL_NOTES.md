## Design Direction For Images

- Product type: AI procurement/compliance review workspace.
- Audience: business users, workshop judges, procurement teams, founders, reviewers.
- Tone: premium, calm, clear, trustworthy, modern.
- Visual world: refined command center, light editorial compliance, precise data cards, subtle dark accents.
- Palette proposal:
  - deep ink: `#101214`
  - warm paper: `#f7f4ec`
  - soft mist: `#edf2f4`
  - teal lens: `#087b78`
  - cobalt evidence: `#285ed8`
  - amber risk: `#bd750f`
  - danger red: `#b42318`
- Typography proposal:
  - refined grotesk for UI
  - optional editorial display feel for large headings
- Avoid:
  - purple-blue generic AI gradients
  - cluttered card walls
  - technical jargon in visible labels
  - tiny unreadable text
  - logo marks that are only a clipboard/check icon
